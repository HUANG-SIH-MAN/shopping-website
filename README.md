#### 環境建置與需求

- 使用框架 express
- 使用樣板引擎 express-handlebars
- 使用資料庫 MySOL
- 安裝
  - 下載專案
    ```
    https://github.com/HUANG-SIH-MAN/shopping-website.git
    ```
  - 安裝專案
    ```
    $ cd shopping-website
    $ npm install
    ```
  - 資料庫建置
    ```
    $ npx sequelize db:migrate:all
    $ npx sequelize db:seed:all
    ```
  - 執行程式
    ```
    $ npm run start
    $ npm run dev
    ```
  - 伺服器位置
    ```
    localhost:3000
    ```
